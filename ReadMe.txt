Irkutsk_model_parameter_tuning - переборочный алгоритм для подбора наилучших параметров модели градиентного бустинга под данные по иркутстку ('irkutsk.csv')

Shap_data_creation - Сама модель машинного обучения. При запуске дает вам выбрать один из 4 файлов для обучения. После обучения позволяет сохранить статистику для дальнейшей оценки факторов используемых моделью.
Правки в него вносить не надо. Потом возможно понадобится дописать сохранение весов.

shap_statistics_visualization_instrument - Позволяет визуализировать данные от "Shap_data_creation.py". Перед использованием убедитесь, что вы сохранили данные в "Shap_data_creation.py", иначе она будет использовать данные с прошлого сохранения.
При использовании можете подстроить последние строчки под себя:
Summary_plot(shap_values, X) - можно убрать или закомментить
Bar_plot(shap_values, X) - можно убрать или закомментить
#Dependence_plot(shap_values, X, 0) - число в конце - это номер фактора, по которому строится график. Обычно это целое число от 0 до 39.
